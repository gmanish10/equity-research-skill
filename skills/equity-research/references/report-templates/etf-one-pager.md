# ETF one-pager — deliverable template

One page, dense, skimmable.

## Structure

### Header
- `{ETF Name} ({Ticker})`
- Sponsor | Inception | AUM | Expense Ratio
- One-line verdict (top-right): e.g., "SATELLITE — BUY, conviction MEDIUM"

### Snapshot
- What it tracks
- Methodology (cap-weighted / equal-weighted / smart-beta / active)
- Top 10 holdings + % in top 10
- Sector breakdown (top 5 sectors)
- Country exposure (if international)

### Key metrics table
| Metric | Value |
|---|---|
| 1Y total return | X% |
| 3Y annualized | X% |
| 5Y annualized | X% |
| Max drawdown (5Y) | -X% |
| Sharpe (5Y) | X |
| Correlation with SPY | 0.XX |

### Factor exposure
Bullet list of primary factor tilts.

### Role in portfolio
One paragraph — what purpose does this ETF serve? Does it complement existing holdings?

### Rating block
```
Rating:          [Buy / Hold / Avoid]
Role:            [core / satellite / thematic / hedge / avoid]
Conviction:      [High / Medium / Low]
Entry approach:  [lump sum / DCA / on pullback to $X]
Position size:   [X%]
WHAT KILLS THIS TRADE: [specific risks]
```

### Risks
3–4 bullets: concentration, liquidity, methodology quirks, sponsor, tax.

### Disclaimer (footer)
Mandatory.
